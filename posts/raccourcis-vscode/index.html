Les raccourcis VS Code

31.01.2023 — Écrit par Salomé Gfeller
#article  #vscode 

Publié dans le “Frontend Weekly” d’il y a quelques semaines, cet article sur “Medium”, présente plusieurs raccourcis claviers utiles sur VS Code.

Pourquoi #

Pouvoir programmer rapidement et sans prise de tête (quand il s’agit du setup) est important pour pouvoir se concentrer sur le code et les problèmes à résoudre.

Résumé #

Ces raccourcis claviers évitent de devoir quitter le clavier et utiliser la souris. (Je vais présenter ceux pour Mac)

  • Cmd + Opt + ⬆️ / ⬇️ pour créer plusieurs curseurs
  • Cmd + D pour sélectionner la prochaine occurence de ce qui est actuellement sélectionné
  • Cmd + U pour désélectionner ce qu’on a fait au raccourcis précédent
  • Ctrl + Fn + ⬆️ / ⬇️ pour scroller sans bouger le curseur
  • Opt + ⬅️ / ➡️ pour bouger le curseur un mot à la fois (au lieu d’un charactère à la fois)
  • Opt + Shift + ⬅️ / ➡️ pour bouger le curseur un mot à la fois en le sélectionnant
  • Cmd + ⬅️ / ➡️ pour sauter au début ou fin de la ligne de code
  • Cmd + Shift + ⬅️ / ➡️ pour séctionner la ligne depuis le début ou la fin
  • Ctrl + Shift + ⬅️ / ➡️ pour sélectionner en incluant plus ou moins du scope (string, fonction, etc.)
  • Cmd + J pour afficher sur le terminal, Cmd + 1 pour focus sur le code
  • Ctrl + Cmd + Opt + * pour diviser le terminal en deux
